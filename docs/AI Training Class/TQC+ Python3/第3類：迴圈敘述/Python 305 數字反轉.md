#Python #String #TQC 
#### 題目說明：

請撰寫一程式，讓使用者輸入一個正整數，將此數值以反轉的順序輸出。

#### 範例輸入1

```
31283
```

#### 範例輸出1

```
38213
```

#### 範例輸入2

```
1003120
```

#### 範例輸出2

```
0213001
```

---
#### Solution
```python linenums="1"
a = input()
print(a[::-1])
```
