### 如何使用別人寫好的專案
1. 確定使用別人專案的原因
   1. 裡面有自己想要的介面樣式（但不會寫）
   2. 想直接套用版面，在改一些參數值
2. 使用時由於每個人建立的檔案或多或少會有缺漏，例如：
   1. API Key缺少
   2. 圖片沒有
   3. 過往的網址消失了
   4. **使用的package與你自己的環境版本不相符**
   5. Flutter的SDK已經改版了，不支援了
....

由於以上種種原因，都可能使你好不容易下載完的Flutter專案檔案編譯不了，因此最好的方式=>`將程式碼直接抓進lib資料夾，並在pubspec.yaml加入所需package`
