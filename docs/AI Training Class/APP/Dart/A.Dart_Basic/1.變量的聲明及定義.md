### 定義
Q：什麼是變量？
A：變量代表可以改變的數值，而這個數值被裝在容器裡，這個容器可能是可變得也可能是不可變的。
### 使用變量
```dart
main(){
	var name = "Ahwu";
}
```
上述程式碼可以看出將`name`設定為`String`資料格式，**若變量的資料格式被固定過後便不能被改變，否則將產生錯誤。**
以下賦值將使變量數值默認為null
```dart
String value;
```
`value`內容為**null**
### 不可變變量
- 使用final
- 使用const
聲明時必須賦值，如果在聲明時沒有對其進行賦值或者在定義完成後又對其進行了修改，在運行時就會拋出異常。
