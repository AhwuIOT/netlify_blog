算數運算符通常用來進行簡單的數據運算，例如加、減、乘、除等。
四則運算是數學中基礎的運算類型。 “+”用來進行相加運算。在Dart中，並非只有數值才支持加運算，字符串、列表、Map等數據類型也支持進行相加運算，例如：
```dart
print(1+2);//3
print("1"+"2");//12
print(3-1);//2
//1，當符號“-”只有一個操作數時，實際上是負號運算符，即將正數變成負數，負數變成正數
print(-(-1));
print(2*4);//8
print(10/2);//5.0
//如果要進行整除運算，就可以使用“~/”運算符
print(9~/2);//4
print(9%2);//1
var a = 3;
print(a++);//印出3 這時a變成了4，先印後加
print(++a);//印出5 這時a變成了5，先加後印
a++;
print(a);//4
++a;
print(a);//5
```
